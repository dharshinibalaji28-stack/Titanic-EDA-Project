# Titanic Exploratory Data Analysis Project

## Objective
To analyze the Titanic dataset and discover patterns, trends, and relationships.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed

- Dataset overview
- Statistical summary
- Missing value analysis
- Survival distribution
- Gender-based analysis
- Passenger class analysis
- Correlation analysis

## Key Insights

- Female passengers had higher survival rates.
- First-class passengers had better survival chances.
- Passenger features affected survival outcomes.

## Outcome

Developed skills in data exploration, visualization, and analytical thinking.
